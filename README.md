## Repository Structure

This repository is organized by week, following the structure of the course 
as outlined on Brightspace:

### Completed Weeks

- `Week_01_Introduction_to_Spatial_Data/` - Introduction to spatial data analysis
- `Week_02_Coordinate_Systems/` - Working with coordinate reference systems

*Additional weeks will be added as the course progresses*

### Folder Organization

Each week's folder contains:
- `DATA/` - Data files used for that week's exercises
- `.qmd` files - Quarto documents with code and analysis
- `.html` files - Rendered HTML outputs of the Quarto documents

## Data Sources

Course materials and datasets are provided via Brightspace and include:
- Sample spatial datasets for exercises
- Real-world ecological and environmental data
- Publicly available geospatial resources

## How to Use This Repository

### For Instructors/Reviewers
All materials are organized chronologically by week. Navigate to the 
appropriate week folder to review exercises and outputs.

### For Personal Use

Clone this repository to work locally with RStudio:
1. Open RStudio
2. File → New Project → Version Control → Git
3. Paste repository URL
4. Choose local directory
5. Click "Create Project"
